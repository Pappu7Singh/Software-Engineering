# Software Engineering

## 📖 Overview

This repository contains study materials, notes, assignments, projects, diagrams, and documentation related to **Software Engineering**. It serves as a central place to learn and practice software development methodologies, system design, project management, testing, and maintenance concepts.

## 🎯 Objectives

* Understand software development life cycle (SDLC).
* Learn software engineering principles and best practices.
* Design and document software systems.
* Apply requirement analysis and system modeling techniques.
* Explore software testing, maintenance, and project management.

## 📚 Topics Covered

* Introduction to Software Engineering
* Software Development Life Cycle (SDLC)
* Agile and Scrum Methodologies
* Requirements Engineering
* Software Requirements Specification (SRS)
* System Analysis and Design
* Data Flow Diagrams (DFD)
* Entity Relationship Diagrams (ERD)
* UML Diagrams
* Software Architecture
* Design Patterns
* Software Testing
* Project Management
* Software Maintenance and Evolution
